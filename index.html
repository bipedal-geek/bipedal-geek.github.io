Introduction

This is a Robotic Project for Electrical Engineering major Electronics Course. Our project is inspired from the trends of robotics and concept of singularity mentioned by Dr. Dr.Illah Reza Nourbakhsh in his Guest Lecture at ITU and his Book(Robot Futures by Dr.Illah Reza Nourbakhsh). This project motivation is to make a bipedal walking robots. As this is the most difficult field to robotics to deal with that's why we always witness wheel based robotic cars. This project is implementation and extension of Project Biped

Date: July, 2015

More Details

Purpose

Our purpose to build this robot to work on bipedal robotic locomotion and also statically balanced robot. There are two purposes of these projects

  • Bi-Pedal Robotic Locomotion

    • Bi-pedal robotic locomotion is one of the most difficult tasks to deal in robotics. That’s why we are able to see 4-wheel robots. So we built this robot to add our work in this field. And provide a easy DIY platform to work further on such robots.

  • New Robotics Spectrum

    • We used the idea of baby walking i.e. static balancing and developed a easy DIY platform. This is to change the spectrum of robotics in Pakistan. As there are many problems that can’t be solved without bi-pedal robots like walking up stairs.

Working

Working of this robot is differentiated into three main parts; Joint Actuation using muscle, movement algorithm and static balancing. The robot walks using 8 Servos. These servos provide 8 degrees of freedom to robot.

  • We have statically balanced robot by applying spline interpolation for motion.
  • We actuated motion using high torque servo motors.
  • Robot has a vertical clearance apex of 0.68" and a step length of 0.85“
  • All motors are controlled using self-designed Decade Counter based servos controller. Which helps to control all motors with one signal pin.
  • Robot can navigate autonomously and avoid obstacle using ultrasonic sensor.

Components Used

Following are the components used for making this robot.

  • 3D printed parts
  • Ultrasonic Sensor (HC-SR04)
  • Servo motors MG996R (10KG Torque)
  • Arduino UNO with Custom build motor controller
