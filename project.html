Project Description

Purpose

Our purpose to build this robot to work on bipedal robotic locomotion and also statically balanced robot.

  • Bi-Pedal Robotic Locomotion

  • Bi-pedal robotic locomotion is one of the most difficult tasks to deal in robotics. That’s why we are able to see 4-wheel robots. So we built this robot to add our work in this field. And provide a easy DIY platform to work further on such robots.

  • New Robotics Spectrum

  • We used the idea of baby walking i.e. static balancing and developed a easy DIY platform. This is to change the spectrum of robotics in Pakistan. As there are many problems that can’t be solved without bi-pedal robots like walking up stairs.

Required Components

Following is the list of components needed for this project:

  • 3D printed parts
  • Ultrasonic sensor (HC-SR04)
  • Servo motors MG996R (10KG Torque)
  • Arduino UNO with Custom build motor controller
  • 7.4V Lipo batteries
  • A lot of jumper wires
  • Wire cutter/stripper

Description

Working of this robot is differentiated into three main parts; Joint Actuation using muscle, movement algorithm and static balancing. The robot walks using 8 Servos. These servos provide 8 degrees of freedom to robot. Joint muscle actuation is done using servo motor which work on the movement data extracted from spline interpolation. Spline interpolation is applied on the already provided data sets for DOF of each joint along with apex and step length. We have developed a decade counter which controls 8 servo motors synchronously. This shield helps to control all these servos from two aurdino PWM pins.

  • We have statically balanced robot by applying spline interpolation for motion.
  • We actuated motion using high torque servo motors.
  • Robot has a vertical clearance apex of 0.68" and a step length of 0.85“.
  • All motors are controlled using self-designed Decade Counter based servos controller. Which helps to control all motors with one signal pin.
  • Robot can navigate autonomously and avoid obstacle using ultrasonic sensor.

Walking and Obstacle Avoidance Code

This code contains walking code for pre defined step frames using Arduino UNO & 4017 Based Custom Designed Servo Controller.. 

/////////////////////////////////////////////////////////////////////////////////////////////////////////////
void loop()
////////////////////////////////////////////////////////////////////////////////////////////////////////////
{
  //this is the main update loop for the microcontroller
  
  //update the current action
  UpdateAction();
  
  //get the frame from the current action  
  int frame[numberOfServos];
  action->GetCurrentFrame(frame);
  
  //set the servo positions for this frame number
  SetServoPositions(frame);                    
}

/////////////////////////////////////////////////////////////////////////////////////////////////////////////
void SetServoPositions(int* frame)
/////////////////////////////////////////////////////////////////////////////////////////////////////////////
{
  //update each servo
  for (int servo = 0; servo < numberOfServos; servo++)
  {   
    //the servo angle used is the linear interpolation between the two 
    float servoAngle = (frame[servo] + calibration[servo])/100.0f;
     
    // the servo control shield commands the servos via pulse width modulations, not angles
    // the PWMs range from 1000 (equal to 0 degrees) to 2000 (equal to 120 degrees)
    // so the servo angle needs to be converted to the corresponding PWM range.
    int pwm = (int)(servoAngle/120.0* (float)(maximumServoShieldPosition - minimumServoShieldPosition)) + minimumServoShieldPosition;
      
    //Set PWM for the servo shield to send out to the servo
    servos.setposition(servo, pwm);       
  }
}

//////////////////////////////////////////////////////////////////////////////////////////////////////////
bool ObstacleInPath()
//////////////////////////////////////////////////////////////////////////////////////////////////////////
{
  //initialize the result
  boolean foundObstacle = true;
 
  //measure the distance 
  distance = MeasureDistance();
  
  //ignore error readings 
  if(distance == 0)
    distance = lastDistance;

  //make sure the object was observed at least twice in a row (high frequency filter)
  if(distance > 5 && lastDistance > 5)    
    foundObstacle = false;
        
  //remember the distance 
  lastDistance = distance;
    
  //return the result
  return foundObstacle;
}
